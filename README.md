# let's catch up please!! 🌍

My friends and I live in different timezones and I've forgotten catch up times because I mixed up the timezones.

So this tool is to help with that.

---

## what it does

Pick a date and drag the time slider — three clocks update in real time showing what that moment looks like in **London**, **Kuala Lumpur**, and **Toronto**. click any clock to make it the controlling timezone, so you can think in your friend's local time instead of doing maths in your head at midnight.


## built with

- vanilla HTML, CSS, and JavaScript — no frameworks, no build step
